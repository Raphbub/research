Ajout des coordonnées X et Y des centroides de bâtiments

# Fonction d'extraction des coords d'un centroide
# Prend une géométrie et une coordonnée (1 pour x, 2 pour y)
extract_cent_coord <- function(geom, coord){
  coords <- st_coordinates(st_centroid(geom))
  return(round(coords[coord], 1))
}

# Ajustement du dataframe pour avoir id, x, y aux 3 premières colonnes
buildings_coords <- buildings_import %>%
                    rowwise() %>%
                    mutate(x = extract_cent_coord(geom, coord = 1),
                           y = extract_cent_coord(geom, coord = 2)) %>%
                    select(id, x, y, everything())

K-Means sur attributs

On réalise plusieurs Kmeans afin de déterminer quel K vaut plus la peine

wss_clusters = seq_len(40)
for (i in 1:40) {
    d = kmeans(blds_an, i, nstart=5, iter.max = 50)
    wss_clusters[i] = d$betweenss / d$totss
}
plot(wss_clusters, type="o")

On réalise donc un kmeans avec k = 7
